# 📝 To-Do App (C++)

## 📌 Description

A simple console-based To-Do application built using C++ with file handling.

## ✨ Features

* Add tasks
* View tasks
* Mark tasks as complete
* Delete tasks
* Data stored in file

## 🛠️ Tech Used

* C++
* File Handling
* STL (vector)

## ▶️ How to Run

1. Compile:
   g++ main.cpp -o todo
2. Run:
   ./todo
