- Delete Student
- Search Student
- Display Students

## 🛠️ Tech Used
- C++
- Data Structures (Linked List)

## ▶️ How to Run
1. Compile using g++
2. Run the executable

## 📚 Concepts Used
- Structures

- Linked List
